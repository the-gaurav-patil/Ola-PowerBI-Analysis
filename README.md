# 🚖 Ola Ride Data Analysis Dashboard (Power BI)

## 📌 Project Overview
This project analyzes Ola ride data using Power BI to uncover key insights related to bookings, revenue, cancellations, and customer behavior.

The dashboard provides a complete business view to help improve operational efficiency and decision-making.

---

## 🎯 Problem Statement
To analyze Ola ride data and identify:
- Booking trends over time  
- Revenue distribution  
- Cancellation reasons  
- Customer & driver behavior  

---

## 🛠️ Tools Used
- Power BI  
- Excel / CSV Dataset  
- Data Visualization  

---

## 📊 Dashboard Features

### 🔹 Overall Analysis
- Total Bookings: **56K+**
- Total Revenue: **19M+**
- Success Rate: ~62%
- Ride trends over time  

### 🔹 Cancellation Analysis
- Total Cancelled Rides: **19K+**
- Cancellation Rate: **28%**
- Major Reasons:
  - Driver not moving
  - Driver asked to cancel
  - AC issues / wrong address  

### 🔹 Revenue Analysis
- Highest revenue from:
  - **Cash & UPI payments**
- Credit/Debit usage is very low  

### 🔹 Vehicle Type Analysis
- All vehicle types generate similar revenue (~8–9M)
- **Auto has lowest average distance**
- Bikes & E-Bikes show good usage efficiency  

### 🔹 Ratings Analysis
- Driver & Customer ratings are consistent (~4.0)
- Indicates stable service quality
